TroubleShooting

The complexity involved in installation of bioinformatics pipelines arise from the hetereogenity of the various systems it integrates. This variety stems from disparate database schemas, program versions and operating environment. One main aim of biopipe is to make these things explicit so that it becomes clear to the user and others what these parameters are and will hopefully make for better appreciation of work processes. This particular document seeks to address some of the problems that one may face in the course of installing and using Biopipe.

Let us know!

Please let us know if you meet any difficulties in the course of using Biopipe. We also appreciate any feedback of any workarounds that you might have to do for your specific environment. It is hard to pre-empt various hardware and software quirks and so we need your help for that.
